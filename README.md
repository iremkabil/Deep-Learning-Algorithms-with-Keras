# Deep Learning Algorithms with Keras

This repository contains implementations of fundamental and advanced deep learning algorithms using Python and Keras.  
Each model was developed as part of a structured training program focused on practical applications of deep learning.

The goal of this project is to understand different neural network architectures and apply them to real-world datasets.
